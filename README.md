# 🎴 Flashcard Generator

Ứng dụng học từ vựng tiếng Anh với các tính năng:

## ✨ Tính năng chính

- **📚 Duyệt thẻ**: Xem tất cả flashcard với hiệu ứng lật thẻ
- **📖 Học tập**: Học từ vựng theo thứ tự với theo dõi tiến trình  
- **🎯 Kiểm tra**: Quiz trắc nghiệm với đếm ngược thời gian
- **✍️ Điền từ**: Luyện viết chính tả với gợi ý

## 🚀 Cách sử dụng

1. Nhập JSON từ vựng vào ô nhập liệu
2. Bấm "Tạo Flashcards" để tạo thẻ
3. Chuyển giữa các chế độ học tập

## 📋 Format JSON

```json
[
  {"word": "example", "definition": "ví dụ", "partOfSpeech": "n", "pronunciation": "/ɪɡˈzæm.pəl/"}
]
```

## 🛠️ Công nghệ

- HTML5
- CSS3 (với CSS Variables, Glassmorphism)
- JavaScript (Vanilla JS, Web Speech API)
- LocalStorage để lưu trữ

## 📱 Responsive

Giao diện tương thích với mọi kích thước màn hình.
